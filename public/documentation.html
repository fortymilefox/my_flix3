Documentation

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of ALL movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing a title, description, genre property, director property and image.
Get data about a movie genre /genre/[Name] GET None A JSON object holding data about a movie genre. Example:

{ "Name": "Genre of movie: e.g. Comedy", "Description": "Desc. of genre" }

Get data about a director /directors/[Name] GET None A JSON object holding data about a director. Example:

{ "Name": "Name Here", "Bio": "Short Biography here", "DOB": "DOB Here" }

Add new user /users POST A JSON object holding data about the user to add, structured like:

{ "Username" : "Username Here", "Password" : "Password Here" "Email" : "Email Here" "Date of Birth" : "DOB Here" }

A JSON object holding data about the user that was added, including an ID:

{ "Favorites": [], "_id": "ID Here", "Username" : "Username Here", "Password" : "Password Here" "Email" : "Email Here" "Date of Birth" : "DOB Here" }

Update a user's information /users/[Username] PUT A JSON object holding data about the user to add, structured like:

{ "Username" : "Username Here" "Password" : "Password Here" "Email" : "Email Here" "Date of Birth" : "DOB Here" }

A text message indicating the user's information was updated
Add movies to user's list of favorites /favorites/[Username]/[Title] POST None Message indicating the movie was added to the user's list of favorites
Remove movie from user's list of favorites /favorites/[Username]/[Title] DELETE None A message indicating the movie was removed from the user's list of favorites
Remove user accounts /users/[Username] DELETE None A message indicating the user's account was deleted
